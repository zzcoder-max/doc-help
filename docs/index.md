# Gemini 验证经验指导

开始之前， 让我们先感谢 Google 老板大气！心诚则灵，毕竟一会会有一些玄学的东西存在(其实是黑盒的风控策略)。 

## 准备工作

在开始验证之前，你一定要准备的内容： 1 和 3 暂不提供，需要的人多会单独再建商品

1. 纯净的 google 账号 (新注册的账号需要等待一点时间)
2. 纯净的 IP 节点，（ 如果你没有， 可以找我）
3. visa 卡之类海外支付方式


OK, 确认你有了这些内容之后， 请不要着急去到学生认证网站， 先检查几个内容: 

1. 修改账号年龄 18 ~ 25 岁， 123 到个人中心修改
<img width="598" height="532" alt="image" src="https://github.com/user-attachments/assets/495ddb83-7627-4d60-bf39-eeafc9278ef7" />
2. 修改地区为美国， 可以是具体的城市比如: Dover
3. 绑定手机号
4. 重要: 检查账号地区， 最好是美国，很多城市会被禁止，比如 Hongkong。 如何查看看下图, 更换访问链接 https://policies.google.com/country-association-form?hl=en-TW&fg=1

## 开始认证

上面的如果都有了，并且你的账号没有经过多次验证，那你的成功率已经相对高了很多了， 接下类我们继续: 

1. 打开 google 的资格页面: https://one.google.com/ai-student, 【之后先不要操作 ！！！】 如果你看到下图，那么你的 ip，账号大概率是没问题的

<img width="692" height="692" alt="image" src="https://github.com/user-attachments/assets/157fdfb3-ebe8-45ec-bfa2-1e88296292e8" />

2. 你没有操作，很好， 接着用右键点击按钮 (如果是 mac, 二指)， 点击复制链接

<img width="740" height="753" alt="image" src="https://github.com/user-attachments/assets/f189da23-4a74-4f6c-aade-d7b9c8956560" />

3. 把链接发给我， 如果你之前没有自己尝试做过 sheerid 认证并且失败的经验， 那么应该问题不大

4. 验证通过后，你将看到这个页面， 开始订阅即可
